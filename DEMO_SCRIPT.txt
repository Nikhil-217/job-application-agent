[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png) (pause)
Let me walk you through the system architecture. First, any input pasted by the user enters the Security Checkpoint, where it is sanitized and validated. From there, the Orchestrator Agent manages the conversation flow and state transitions. It delegates to the Job Analysis Agent to parse requirements, which then feeds into the Resume Tailor Agent. The Resume Tailor Agent calls the local Resume MCP Server to read the original resume and rewrites key bullet points to match ATS keywords. Next, the Cover Letter Agent drafts a customized cover letter, and a Tracker Agent uses the Sheets MCP Server wrapper to log details. The final output is updated with versioned files and a Follow-Up Agent drafts email templates. All of this includes a built-in Human Approval pause to keep you in control at every single step.

[3:00 — SECURITY & MCP]
(pause)
What makes this system unique is its security and integration design. The security checkpoint rate limits requests, sanitizes inputs, and redacts PII like phone numbers or Aadhaar cards before logging. Meanwhile, our custom Sheets MCP server wrapper redirects logging to stderr, preventing the protocol crashes common in standard implementations.
